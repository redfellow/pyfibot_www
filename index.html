Pyfibot

A Python IRC-bot made using the Twisted Matrix IRC-library.

Supports online module reloading - only major core changes require a restart. Extensive module & handler -support for easy extension and customization. Check the Installation instructions and the module HOWTO.

Jump right in!

Clone

To start, grab the Pyfibot repository url from Github. Refer to Installation instructions and preconditions for more detailed info.

Setup

Get your environment configured for easy deployment. There are a few different ways to do this, choose what suits you the best!

Connect

Finish the deal by editing the example configuration and launching the bot!
